# Odora — رایحه‌ات رو با ما کشف کن 🌸

**اُدورا** یک وب‌سایت پیشنهاددهنده‌ی عطره: کاربر به چند سؤال حسی و کاربردی درباره‌ی سلیقه‌اش جواب می‌ده و سایت از بین **۱۰۰ عطر معروف دنیا**، پنج عطری که بیشترین تطابق رو با سلیقه‌ش دارن پیشنهاد می‌ده — همراه با **درصد نزدیکی** هر عطر به سلیقه‌ی کاربر.

---

## ✨ امکانات ساخته‌شده تا الان

| بخش | توضیح |
|---|---|
| **صفحه‌ی اصلی** | معرفی سایت + دکمه‌ی «شروع تست» + بخش «چطور کار می‌کنه» |
| **انتخاب سبک** | زنانه / مردانه / یونیسکس — رنگ‌بندی صفحات تست با انتخاب کاربر عوض می‌شه |
| **تست بویایی** | ۶ سؤال: فضای موردعلاقه، لحظه، امتیازدهی به ۷ خانواده‌ی بویایی، موقعیت استفاده، فصل، قدرت بو |
| **نتایج** | ۵ عطر برتر با درصد تطابق + حلقه‌ی درصد + لینک اشتراک‌گذاری نتیجه (بدون نیاز به سرور) |
| **صفحه‌ی هر عطر** | نمودار پروفایل بویایی، فصل/موقعیت/قدرت، دکمه‌ی خرید (به‌زودی) |
| **گنجینه‌ی عطرها** | مرور هر ۱۰۰ عطر با تب‌های جنسیت + **جستجوی نام/برند** + فیلتر **فصل / خانواده‌ی بویایی / قدرت بو** (همه با هم ترکیب می‌شن) |
| **علاقه‌مندی‌ها ♥** | دکمه‌ی قلب روی کارت‌ها و صفحه‌ی هر عطر؛ صفحه‌ی «علاقه‌مندی‌های من» از آیکون قلب در هدر — بدون نیاز به ثبت‌نام، ذخیره در مرورگر |
| **عطرهای مشابه** | پایین صفحه‌ی هر عطر، ۴ عطر هم‌جنسیت با نزدیک‌ترین پروفایل بویایی (با همان الگوریتم تطابق) |
| **حساب کاربری (محلی)** | ورود/ثبت‌نام با نام و ایمیل از آیکون کاربر در هدر؛ پروفایل: تعداد علاقه‌مندی‌ها + آخرین نتیجه‌ی تست + خروج. فعلاً روی دستگاه ذخیره می‌شه و در فاز بک‌اند به سرور وصل می‌شه |
| **لینک هدیه 🎁** | در صفحه‌ی هر عطر: «هدیه دادن این عطر» با اسم و پیام اختیاری → لینک کپی می‌شه؛ گیرنده صفحه‌ی هدیه‌ی اختصاصی با رنگ همان عطر می‌بینه — بدون سرور |
| **درباره ما / تماس** | صفحات اطلاعاتی + فرم تماس |
| **حالت روز/شب** | دکمه‌ی خورشید/ماه در هدر — انتخاب کاربر ذخیره می‌شه |
| **دو زبانه: فارسی + انگلیسی** | دکمه‌ی زبان در هدر (EN / فا) — کل سایت با یک کلیک انگلیسی می‌شه: متن‌ها، جهت صفحه (چپ‌به‌راست)، فونت (Poppins)، اعداد لاتین و حتی توضیح ۱۰۰ عطر. انتخاب زبان ذخیره می‌شه |
| **تم‌های رنگی** | طلایی (عمومی/یونیسکس)، رزگلد (زنانه)، سرمه‌ای/استیل (مردانه) — هر کدوم نسخه‌ی تیره و روشن دارن |

### قوانین تم (مهم)
- صفحات عمومی (خانه، گنجینه، درباره ما، تماس) → همیشه تم **طلایی**
- مسیر تست (انتخاب سبک، تست، نتایج) → تمِ **جنسیت انتخاب‌شده**
- صفحه‌ی جزئیات عطر → تمِ **جنسیت خودِ اون عطر**
- پس‌زمینه‌ها همیشه خنثی هستن؛ رنگِ تم فقط روی دکمه‌ها و اکسنت‌هاست (چشم رو اذیت نکنه)
- هر بخش **فقط عطرهای همون جنسیت** رو نشون می‌ده (فیلتر سخت‌گیرانه)

## 🚀 اجرای سایت

**پیش‌نیاز: فقط پایتون** (Python 3) — چیز دیگه‌ای لازم نیست؛ نه Node.js، نه نصب پکیج.

**راه آسون:** روی فایل `run.bat` دابل‌کلیک کن → سرور روشن می‌شه و مرورگر خودش باز می‌شه.

**راه دستی:** توی همین پوشه یک ترمینال باز کن و بزن:
```
python serve.py
```
بعد توی مرورگر برو به: **http://localhost:5500**

> ⚠️ فایل `index.html` رو مستقیم (دابل‌کلیک) باز نکن — چون سایت از ماژول‌های جاوااسکریپت و فایل ترجمه استفاده می‌کنه، حتماً باید از طریق سرور (همون `serve.py`) باز بشه.

**انتشار روی اینترنت:** سایت کاملاً استاتیکه (بدون بک‌اند)، پس روی هر هاست استاتیکی مثل GitHub Pages، Netlify یا Vercel رایگان بالا میاد — فقط کل فایل‌ها رو آپلود کن.

## 💾 بک‌آپ گرفتن

**راه آسون:** روی `backup.bat` دابل‌کلیک کن → یک فایل zip با تاریخ روز، روی **دسکتاپ** ساخته می‌شه.

پروژه **گیت (Git)** هم داره — یعنی تاریخچه‌ی تغییرات داخل خودش ذخیره می‌شه. بعد از هر تغییر مهم می‌تونی توی ترمینال بزنی:
```
git add -A
git commit -m "توضیح تغییر"
```

فایل **`CLAUDE.md`** توی ریشه‌ی پروژه، راهنمای مخصوص Claude ــه: ساختار پروژه، قوانین تم، قوانین کار (مرحله‌به‌مرحله با تأیید تو) همه توش نوشته شده.

- **Claude Code (پیشنهادی):** پوشه‌ی پروژه رو باز کن و ادامه بده — خودش `CLAUDE.md` رو می‌خونه.
- **پروژه در claude.ai:** یک Project بساز و فایل‌های `CLAUDE.md` و `README.md` رو (و در صورت نیاز فایل‌های کد رو) به بخش Knowledge اضافه کن.

## 📝 تاریخچه‌ی کارهای انجام‌شده

1. **فاز ۱–۸ (نسخه‌ی اول):** ساختار پروژه، سیستم ترجمه، روتر، صفحات اصلی، تست ۶ مرحله‌ای، دیتابیس ۱۰۰ عطر، الگوریتم تطابق، نتایج با لینک اشتراک، گنجینه، درباره/تماس
2. **بازنگری ۱:** تم صفحه‌ی اصلی همیشه طلایی، دکمه‌ی روز/شب، هر تم دو حالته شد، رنگ مردانه مردونه‌تر (سرمه‌ای/استیل)، گنجینه به منو اضافه شد + تب‌های فیلتر، فیلتر سخت‌گیرانه‌ی جنسیت در نتایج، حذف دکمه‌ی اضافه از درباره ما
3. **بازنگری ۲:** پس‌زمینه‌ی تم زنانه خنثی شد — رزگلد فقط اکسنت (رفع خستگی چشم)
4. **قابلیت جدید — زبان انگلیسی:** دکمه‌ی تغییر زبان در هدر؛ ترجمه‌ی کامل رابط + توضیح ۱۰۰ عطر به انگلیسی (`en.json` و `perfumesEn.js`)؛ جهت صفحه، فونت و اعداد به‌صورت خودکار با زبان عوض می‌شن؛ انتخاب زبان در مرورگر ذخیره می‌شه
5. **فاز ۹ — شش قابلیت جدید:** جستجو در گنجینه، فیلترهای فصل/خانواده/قدرت، علاقه‌مندی‌ها (♥ روی کارت‌ها + صفحه‌ی اختصاصی + آیکون هدر)، «عطرهای مشابه» زیر هر عطر، حساب کاربری محلی (آیکون هدر، آماده‌ی اتصال به بک‌اند)، و لینک هدیه با اسم/پیام اختیاری و صفحه‌ی دریافت هدیه — همه دوزبانه (فا/EN)
